# ResumeForge 🚀

An Overleaf-like LaTeX resume editor that tailors your resume to any job posting using Google Gemma 4.

## Features
- 📝 Live LaTeX editor with PDF preview
- 🔍 Job URL parser — extracts requirements, qualifications, sponsorship info
- 🤖 AI-powered resume tailoring via Gemma 4
- 📥 Auto-named PDF downloads (`name_resume_company_date.pdf`)
- 🔐 API key login with validation

## Setup
1. Get a Google Gemini API key from [Google AI Studio](https://aistudio.google.com/)
2. Enter it on the login screen — it's validated live and never stored on the server
3. Paste a job URL, upload your base LaTeX resume, and let Gemma do the work!
